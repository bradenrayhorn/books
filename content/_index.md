# Books!

For a couple years I've been cataloguing the books I read, mainly because I was curious to see
how many words I read annually.

Turns out an accurate word count is really hard to find.
I was using [wordcounters.com](https://wordcounters.com/word-count/conversion.jsp) audiobook length
to word count conversion tool for a while.
Then I found [Rakuten kobo](https://www.kobo.com). They share an estimated word count for (mostly) all books.
Supposedly this is derived from scanning the e-book. I've been using it to populate my word counts since.

After I got my word count situation resolved, I began feeling like I should take some time and really reflect on books after reading. See how it made me feel: Why'd I like it? What'd I dislike? Maybe even write some reviews.
It's not a skill I am good at.

I got the urge to turn this idea into a website. Why? There is no real purpose.
I just like making web apps. So out came this website. It'd be simpler as a spreadsheet, but where's the fun in that?

I catalog every book and series I read. They are ranked on a simple scale from C to AA. C being "okay" and AA being "excellent".
All spoilers are hidden by default and must be revealed.

This site is mostly just for me, but perhaps you can use it to find a book that interests you too :)
